About the Project

SkinAI Classifier is a senior capstone project developed at Purdue Fort Wayne University. Our mission is to create a user-friendly interface for uploading skin images, collecting relevant patient information, and integrating machine-learning models that assist with early detection and classification of skin diseases.

This project is not a medical diagnosis tool - it is purely for research, education, and demonstrating AI-assisted classification workflows.
